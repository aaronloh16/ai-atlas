# Local AI Stack

A privacy-friendly local-first setup for coding and experimentation without always relying on hosted APIs.

## Core stack
- Ollama
- llama.cpp
- Open WebUI
- Qdrant
- Continue

## Suggested architecture
- Local inference runtime: Ollama and/or llama.cpp
- User interface: Open WebUI
- Retrieval store: Qdrant
- Developer workflow: Continue in IDE

## Who this is for
- Developers who want local control, low incremental cost, or offline workflows.

## Swap options
- Replace Continue with Aider for terminal-first coding.
- Add LocalAI when OpenAI-compatible self-hosted APIs are needed.
