# Stacks

Opinionated starter stacks for common AI builder goals.

These are not one-size-fits-all architectures. They are practical starting points you can adapt by team size, latency budget, deployment model, and compliance needs.

## Available recipes

- [AI Coding Stack](./ai-coding-stack.md)
- [Build an Agent](./build-an-agent.md)
- [Local AI Stack](./local-ai-stack.md)
- [RAG Stack](./rag-stack.md)
- [Voice AI Stack](./voice-ai-stack.md)
- [Eval Stack](./eval-stack.md)

## How to use recipes

- Start with the smallest workable stack.
- Replace one layer at a time as constraints become clear.
- Add observability/evals before adding autonomy.
