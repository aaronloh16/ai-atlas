# Reports

Periodic snapshots of AI dev tool adoption, based on public data and community signals.

## Available reports

| Report | Date | Covers |
|---|---|---|
| [March 2026 Snapshot](./mar-2026-tool-sentiment.md) | 2026-03-06 | CLI downloads, IDE installs, GitHub stars, community sentiment |

## What's in a report

Each report includes:
- **Usage numbers** — npm downloads, VS Code marketplace installs, GitHub stars with source links
- **Community signals** — themes from X, HN, and Reddit (clearly flagged as anecdotal)
- **Safety notes** — things to watch out for when using these tools

Data sources are linked inline so you can verify the numbers yourself.

## Cadence

No fixed schedule yet. Reports get written when there's enough new data to be worth a snapshot. Goal is roughly monthly.

## Methodology

Numbers come from public APIs (npmjs, VS Code marketplace, GitHub). Community sentiment comes from manual observation — see [data/](../data/) for the raw claims and their verification status.

See also: [How tools get picked](../research-method.md)
