# Contributing

Thanks for helping out. This repo needs it.

## Adding a tool

Use this format:

```md
## Tool Name
- Link: https://link
- Category: Category Name
- What it is: One-sentence definition.
- Why it matters: Why it's worth knowing about.
- Best for: Who should look at this.
- Best paired with: What it works well with.
```

## Guidelines

- **Link to the real thing** — official docs or primary GitHub repo, not a blog post about it.
- **Keep it short** — if your description is longer than a few sentences, trim it.
- **Put it in the right category** — check the existing folders first.
- **Useful > impressive** — we care about tools people actually use, not ones that just demo well.

## What makes a good addition

- You or someone you know actually uses it
- It does something meaningfully different from what's already listed
- It's actively maintained

## What we'll probably skip

- Dead or abandoned projects
- Tools that are basically identical to something already listed
- Marketing-heavy products with no real community adoption

## Pull request checklist

- [ ] Used the entry format above
- [ ] Put it in the right category
- [ ] Linked to the primary source
- [ ] Kept it concise
